Testing Your Widget

This page demonstrates how to load and test your visualization plugin.

Mock API Mode

The widget below is running with mock data (no NS-node required).

Real API Mode

To test with real data:

  1. Start NS-node: node ns-node/server.js
  2. Reload this page
  3. Widget will connect to http://localhost:3009

Integration

To add to dashboard, copy widget.js to your plugin directory and register:

// In brain-dashboard.html
<script src="path/to/widget.js"></script>
<script>
  const widget = new CustomVisualization({
    name: 'my-widget',
    apiEndpoint: 'http://localhost:3009'
  });
  
  const container = document.getElementById('widget-container');
  widget.render(container);
</script>
