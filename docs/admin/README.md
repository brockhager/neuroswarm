# Admin Documentation

## WARNING! DO NOT USE ADMIN NODE UNLESS GIVEN PERMISSION BY FOUNDER

This directory contains critical administrative documentation for NeuroSwarm's admin node infrastructure. Access to admin nodes is strictly controlled and requires explicit founder authorization. Unauthorized use may result in immediate system lockdown and legal action.

## Documents

### admin-node-design.md
This document provides the complete specification for the Admin Node, detailing its purpose as a founder-only observability and control plane. It covers capabilities including real-time monitoring, governance controls, simulation tools, diagnostics, and audit export functionality.

### admin-node-genesis.md
This document outlines the IP recording and security configuration for admin nodes. It includes setup instructions for IP whitelisting, startup verification processes, monitoring endpoints, and governance logging for admin node activities.
