# neuro-program

Solana program and smart contracts for the NeuroSwarm project.

## Overview

This repository contains the Anchor-based Solana programs that implement the core blockchain logic for NeuroSwarm, including staking, consensus, and reward distribution.

## Development

### Prerequisites

- Rust
- Solana CLI
- Anchor

### Setup

```bash
npm install
anchor build
```

### Testing

```bash
anchor test
```
