# neuro-shared

Shared libraries and utilities for the NeuroSwarm project.

## Overview

This repository contains common TypeScript utilities, types, and helper functions used across NeuroSwarm components.

## Installation

```bash
npm install @neuroswarm/shared
```

## Development

```bash
npm install
npm run build
npm test
```
