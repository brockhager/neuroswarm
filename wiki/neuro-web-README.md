# neuro-web

Web application and frontend for the NeuroSwarm project.

## Overview

This Next.js application provides the user interface for interacting with NeuroSwarm, including personal AI management, proposal submission, and network monitoring.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Governance

NeuroSwarm uses a badge-weighted voting system for decentralized governance:

### How to Vote

1. **Connect Wallet**: Link your Solana wallet to access governance features
2. **Earn Badges**: Contribute to gain voting power (Bronze=1, Silver=3, Gold=5, Diamond=10 votes)
3. **Browse Proposals**: View active governance proposals in the portal
4. **Cast Votes**: Submit blockchain-anchored votes on Solana
5. **Track Results**: Monitor live voting results and outcomes
