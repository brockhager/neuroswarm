# neuro-runner

Validator runner and node software for the NeuroSwarm project.

## Overview

This repository contains the TypeScript/Node.js implementation of validator nodes, indexers, gateways, and other network participants for NeuroSwarm.

## Installation

```bash
npm install
```

## Development

```bash
npm run build
npm start
```

## Configuration

See `config/default.json` for configuration options.

## Running a Node

### Validator

```bash
npm run validator
```

### Indexer

```bash
npm run indexer
```

### Gateway

```bash
npm run gateway
```
