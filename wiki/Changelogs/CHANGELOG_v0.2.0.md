# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2025-11-28

### Added
- **Phase G Complete**: Full ecosystem expansion including orchestration, governance, and plugins.
- **GPU Resource Management**: Native C++ backend now supports CUDA, CoreML, and DirectML with runtime detection.
- **Plugin System**: Extensible architecture for validators and visualization widgets.
- **Governance Dashboard**: Web UI for voting, audit logs, and parameter tuning.
- **Onboarding Automation**: One-click setup scripts (`onboard.ps1`, `onboard.sh`).

### Changed
- **Architecture**: Refactored `server.js` into modular services (Orchestration, Consensus, Governance).
- **Documentation**: Comprehensive update to Wiki with Learning Hub and System Overview.
- **Performance**: Integrated performance profiling and benchmarking CI/CD.

### Fixed
- **Native Shim**: Improved binary detection and error handling for C++ backend.
- **Health Checks**: Fixed orchestration status reporting.

## [0.1.0] - 2025-10-01

### Added
- Initial release of NeuroSwarm Node.
- Basic chat interface and embedding generation.
