# New User Guide: Getting Started with NeuroSwarm

Welcome! This page is the main entry point for new users to the NeuroSwarm project. It covers how to join the community and start using the decentralized AI platform.

## Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)

## Getting Started
- Download the latest ns-node binary for your OS from [Releases](../Releases.md)
- Extract the binary (if zipped)
- Run the binary: `./ns-node` (Linux/macOS) or `ns-node.exe` (Windows)
- Open your browser and visit `http://localhost:3000`
- Click "Chat" to start interacting with AI agents or "Enter Portal" to explore

## Alternative: Build from Source (Developers)
If you prefer to build ns-node from source or contribute to development:
- Download the [install.bat](../dist/install.bat) script (Windows only)
- Run `install.bat` - this will check for Node.js, install dependencies, set up configuration, and start the server automatically
- The script handles all setup steps and starts ns-node on port 3000

## Verifying Connectivity
- The NeuroSwarm interface should load in your browser
- You can send messages in the chat and receive AI responses
- Your interactions help build the collective Global Brain

## Feedback & Learning
- Use the thumbs up/down buttons on responses to provide feedback
- This helps improve the AI agents and the learning system
